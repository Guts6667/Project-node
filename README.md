# Project-node

This is a personal project made to train myself using node.js

## This app is using:

### Backend:

Node | Express

### Frontend:

React | JavaScript | Sass

## Installation

### Backend :

In the terminal, run: `npm install`

### Frontend :

In the terminal, run: `npm start` or `yarn start`

## Tools used:

nodemon: `npm install -g nodemon`

express: `npm install express`
